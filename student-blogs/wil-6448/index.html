Markdown: lenguaje de marcado ligero.

2023-04-20

Markdown es un lenguaje de marcado que permite dar formato a textos de una manera sencilla y fácil de escribir. Su sintaxis se basa en texto plano, lo que lo hace ideal para la escritura de documentos en línea o para la creación de contenido en sitios web.

Git: software de control de versiones

2023-03-24

Git es un sistema de control de versiones de código abierto que se utiliza ampliamente en el desarrollo de software y otros proyectos colaborativos. Fue creado en 2005 por Linus Torvalds, el creador del kernel de Linux, y desde entonces ha sido adoptado por muchos proyectos comerciales y de código abierto

Terminal de Windows.

2023-01-24

Terminal Windows es un emulador de terminal de múltiples pestañas que Microsoft ha desarrollado para Windows 10 y más adelante como sustituto de la Consola de Windows. Puede ejecutar cualquier aplicación de línea de comandos, incluidos todos los emuladores de terminal de Windows, en una pestaña independiente.

GitHub: repositorio de software abierto

2022-12-24

GitHub es un sitio web y servicio en la nube que ayuda a los desarrolladores a almacenar, administrar y colaborar en su código, utilizando el control de versiones Git
